# Excel & Power BI Data Analysis Portfolio

This repository showcases my hands-on practice and learning journey in **Data Analysis, Visualization, and Advanced Excel & Power BI** skills.

## 📁 Project Files

### 1. **Data Visualization Project.xlsx** (Main Project)
The core project in this repository. A complete **sales data analysis dashboard** for a California-based apparel retailer.

**Key Features:**
- Raw transaction data (`Data` sheet) with 2,400+ records
- Multiple Pivot Tables for different insights:
  - Weekly sales trends
  - Product performance
  - Customer gender & order mode analysis
  - Shipping performance (Days to Deliver)
  - Geographic analysis (State & County level)
- Interactive **Dashboard** sheet
- Various charts and KPI summaries

### 2. **What If Analysis.xlsx**
Demonstrates **What-If Analysis** and scenario modeling using Excel tables and formulas.  
Contains monthly and yearly sales projections for board games with sensitivity analysis.

### 3. **Movie Inventory for Excel SUMIFS.xlsx**
Practical example of using **SUMIFS**, **COUNTIFS**, and other conditional aggregation functions.  
Manages a personal movie collection with analysis by format (Blu-ray/DVD), genre, location, and rating.

### 4. **XlookUp.xlsx**
Showcase of the modern **XLOOKUP** function (superior replacement for VLOOKUP/HLOOKUP).  
Includes practical lookup examples across employee databases.

### 5. **VlookUp.xlsx**
Classic **VLOOKUP** examples for learning and reference, including approximate and exact match techniques.

### 6. **Duplicates.xlsx**
Techniques for identifying, managing, and analyzing **duplicate records** using Excel features like:
- Conditional Formatting
- Remove Duplicates
- Pivot Tables for consolidated views

### 7. **Powe Bi tutorial project.pbix**
**Power BI** dashboard project demonstrating data modeling, DAX measures, and interactive visualizations.

## 📌 How to Use

1. Download the Excel files and open them in **Microsoft Excel** (recommended: Excel 365 or Excel 2021+ for best compatibility with XLOOKUP).
2. Open the `.pbix` file in **Power BI Desktop**.
3. Explore the different sheets and dashboards to see real-world applications of data analysis techniques.
